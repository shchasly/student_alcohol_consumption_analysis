# student alcohol consumption analysis

Dataset: Student Alcohol Consumption

Contents: These datasets content columns like school, sex, age, address, famsize, Pstatus, Medu, Fedu, Mjob, Fjob, reason, guardian, traveltime, studytime, failures, schoolsup, famsup, paid, activities nursery, higher, internet, romantic, famrel, freetime, goout, Dalc, Walc, health, absences, G1, G2, G3 (grades).

Kaggle URL: https://www.kaggle.com/uciml/student-alcohol-consumption
